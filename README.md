# React + Vite Project

## Getting Started

This project is built with:

- React
- Vite
- TypeScript
- Tailwind CSS
- shadcn/ui

### Prerequisites

Make sure you have **Node.js** and **npm** installed.

We recommend using [nvm](https://github.com/nvm-sh/nvm#installing-and-updating) to manage Node versions.
